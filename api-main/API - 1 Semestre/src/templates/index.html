Sobre o Projeto

Essa plataforma tem como objetivo democratizar o acesso aos dados de importação e exportação do Brasil, permitindo consultas por produto, país de origem/destino e períodos. Ideal para pesquisadores, estudantes e profissionais da área.

Gráficos

Para visualizar melhor os dados de importação e exportação, criamos gráficos interativos utilizando Python no Google Colab. Eles nos ajudam a identificar padrões, tendências e os principais produtos e países envolvidos nas transações comerciais do Brasil.

Funcionalidades

  • Busca por NCM (Nomenclatura Comum do Mercosul)
  • Gráficos de exportação por região
  • Estatísticas por produto e país
  • Download dos dados
